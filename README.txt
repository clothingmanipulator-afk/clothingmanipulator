THE CLOTHING MANIPULATOR WEBSITE

1. Open index.html in a web browser to preview the website.
2. The company logo is stored at: assets/logo.jpeg
3. Replace each placeholder image in the assets folder with the real product image.
4. Keep these exact file names:
   Pic 1.jpg through Pic 13.jpg
5. The current gallery setup treats Pic 1–Pic 7 as Bath Suits and Pic 8–Pic 13 as Regular Suits.
   To change a category, edit data-category="bath" or data-category="regular" in index.html.
6. Upload index.html and the complete assets folder to the root of your GitHub repository.
7. File names are case-sensitive on GitHub Pages.
